## OneNeo Bank — 顧客・ペルソナ

> OneNeo Bank コンシェルジュ・デモ用の顧客セグメントとペルソナ（架空・説明用）。
> 会社概要: [OneNeo-Bank](OneNeo-Bank.md)。
> ペルソナはすべて架空であり、自由に改訂可能です。

OneNeo Bank は日本国内に約100万人の個人のお客さまを有します。本書のペルソナは
2つの目的で用います。**(a)** デモの物語を牽引し、コンシェルジュの難ケースを検証する、
記憶に残る少数の「スポットライト・ペルソナ」。**(b)** 以下のタクソノミーから生成する、
大規模な検証・評価用の「合成顧客集団」。

- **スポットライト・ペルソナ:** 以下に手作業で作成（詳細なシナリオ付き）。
- **生成された顧客集団:** [personas/personas-catalog.md](personas/personas-catalog.md)
  （閲覧用）および [personas/personas.json](personas/personas.json)（機械可読）。
  [`pipeline/generate_personas.py`](../../pipeline/generate_personas.py) が生成します。

---

## ペルソナ・スキーマ

各ペルソナは同一の項目で記述します。

| 項目 | 意味 |
|------|------|
| 年齢 / 性別 | 年齢帯（18–27・28–37・38–47・48–59・60+）と性別 |
| 職業 / 収入 | 雇用形態、説明用の年収（円）、収入の安定性 |
| 世帯 | 配偶関係、子どもの人数と年齢 |
| 住まい / 車 / 地域 | 住居形態、車の保有、居住地域 |
| OneNeoとの関係 | 保有商品、メイン／サブ利用、利用期間 |
| 家計 | 他の借入、貯蓄の余力 |
| デジタル習熟度 / 態度 / 言語 | アプリ習熟度、満足度、日本語／英語対応要否 |
| ライフイベント | 相談の「きっかけ（なぜ今か）」 |
| **想定される相談** | このペルソナがコンシェルジュに持ち込む1〜2の相談 |

**想定される相談**は、ペルソナを具体的な会話に結びつける、デモにおいて最も重要な項目です。

## ディメンション・タクソノミー

合成顧客集団は以下のディメンションからサンプリングします。重み付けは日本の実態に
即し、かつネオバンク寄り（若年・アプリ習熟・サブ口座利用が多い）に設定しています。
ジェネレータ側で重みを調整すると集団の分布を変えられます。

- **年齢帯:** 18–27・28–37・38–47・48–59・60+
- **性別:** 女性・男性
- **雇用形態:** 正社員・契約社員・派遣社員・パート／アルバイト・個人事業主・経営者・
  公務員・学生・年金受給者・再雇用・無職 — *年齢に応じて条件付け*
- **収入:** 年収レンジ（円）、*雇用形態に応じて条件付け*。安定性 = steady（安定）／variable（変動）
- **配偶関係:** 独身・既婚・離別・死別 — *年齢に応じて条件付け*
- **子ども:** 人数と年齢 — *年齢と配偶関係に応じて条件付け*
- **住まい:** 賃貸・持家（住宅ローンあり）・持家（完済） — *年齢と収入に応じて条件付け*
- **車:** なし・あり — *地域に応じて条件付け*
- **地域:** 東京23区・東京圏・その他都市部・地方
- **OneNeoとの関係:** 保有商品（普通預金／定期預金／カードローン／デビット利用）、
  メイン／サブ利用、利用期間（月数）
- **家計:** 他の借入（なし／リボ残高／その他ローン／住宅ローン）、貯蓄の余力
  （thin：薄い／moderate：中位／healthy：厚い）
- **デジタル習熟度:** 高・中・低 — *年齢に応じて条件付け*
- **態度:** satisfied（満足）／neutral（中立）／frustrated（不満）
- **言語:** 日本語／英語対応要（外国人居住者）
- **ライフイベント:** 就職・貯蓄・結婚・出産・育児費用・教育資金・住宅購入・引越し・
  海外旅行・転職・起業・退職準備・年金生活 — *年齢に応じて条件付け*

> いつでも再生成可能: `.venv/bin/python pipeline/generate_personas.py --count 100`
> （シード固定のため出力は再現可能。`--count` / `--seed` は調整可）。
