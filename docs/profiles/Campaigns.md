## キャンペーン（Campaigns）

> OneNeo Bank コンシェルジュ・デモ用のキャンペーン・プロファイル（架空・説明用）。
> 会社概要: [OneNeo-Bank](OneNeo-Bank.md) ／ 商品概要: [Products](Products.md)。
> 関連商品: [預金](products/Deposits.md) ／ [カードローン](products/Lending.md) ／
> [デビットカード](products/Debit-card.md)。
> 金額・条件・期間などの数値はすべて説明用であり、自由に改訂可能です。

OneNeo Bank のキャンペーンは、いずれも**口座開設やデビットカードの新規申請直後の
早期利用**を促すことを狙いとしています。お客さまはアプリのプッシュ通知でキャンペーンを
知り、通知をタップしてコンシェルジュに相談したり、条件のシミュレーションや申込の
下書き保存を行います。

キャンペーンは次の3カテゴリです。

1. **預金** — 普通預金・定期預金で、一定条件の達成によりキャッシュバック
2. **カードローン** — 新規契約で、一定条件の達成によりクーポン券を献呈
3. **デビットカード** — 新規申請＋利用で、一定条件の達成によりキャッシュバック

### 共通事項
  - **対象者:** 日本国内在住の個人のお客さま（各キャンペーンの対象条件を満たす方）。
    口座開設の要件（年齢・外国籍等）は [預金](products/Deposits.md) を参照。
  - **エントリー:** アプリのキャンペーン画面からエントリー（事前エントリー制）。
    口座開設後、キャンペーン締切日まで、かつ条件達成より前にエントリーが必要。
  - **「取引」の定義:** 「口座開設後まだ取引のない方」とは、口座の取引履歴に記録がない状態。
    デビット利用・口座振替・入出金等は取引履歴に記録されます。
  - **重複適用:** 特典は口座単位で判定・適用。同一のお客さまでも口座が異なれば、
    それぞれで条件を達成した場合に個別に受け取れます。
  - **税務:** キャッシュバックは一時所得の対象となる場合があります。確定申告等の詳細は
    国税庁のご案内等をご確認ください。
  - **特典付与:** 条件達成を確認後、各キャンペーンに記載の時期に普通預金口座へ入金／付与。
  - **併用:** 特典の併用可否は各キャンペーンに記載。
  - **除外:** 反社会的勢力に該当する方、および当社が不適切と判断した方は対象外。
  - 内容は予告なく変更・終了する場合があります。

### CMP-DEP-2026Q3-01 — 口座開設・普通預金応援キャッシュバック
  預金カテゴリ。口座開設後の早期のお取引を後押しするキャンペーンです。

  - **対象者:** 期間中に新規口座を開設した方、または口座開設後まだ取引のない方
  - **対象期間:** 2026年7月1日 ～ 2026年9月30日
  - **達成条件と特典（複数達成可・合計上限あり）:**

  | 達成条件 | キャッシュバック |
  |----------|------------------|
  | 条件1  給与または年金の受取口座に設定し、期間内に振込実績があること | 1,000円 |
  | 条件2  口座開設後60日以内に普通預金を50万円以上預入 | 1,000円 |
  | 条件3  7、8、9月、3か月連続で普通預金の月末残高100万円以上を維持 | 2,000円 |
  | **合計上限** | **4,000円** |

  - **エントリー:** 弊社Mobile Appでの専用フォームにてエントリーを実施。
  - **条件確認タイミング:** 7月31日、8月31日、9月30日にそれぞれの条件の達成状況を判断。
  - **特典付与時期:** 各条件が達成した月の翌月の10日までに、対象口座へ特典を現金として振込を行う。
  - **給与・年金の判定:** 「給与振込」は振込電文に「給与」「給料」「賞与」、「年金振込」は
    「年金給付金（年金信託）」「年金給付金（公的年金）」の文言が含まれるかで判定します。
    受取口座の設定のみでは対象にならず、期間内の振込実績が必要です。
  - **特典の取消:** 条件達成後、転勤・退職等のやむを得ない理由で設定を解除しても、
    付与済みの特典は取り消しません。ただし、不当な条件操作等と当社が判断した場合は、
    関連する特典を取り消すことがあります。
  - **併用:** 同キャンペーンの上記条件は併用可（合計上限の範囲内）。ただし、CMP-DEP-2026Q3-02 とは併用不可。その他のキャンペーンとの併用可否は各キャンペーンの記載に従います。
  - **関連商品:** [預金](products/Deposits.md)

### CMP-DEP-2026Q3-02 — 口座開設・定期預金支援キャッシュバック
  預金カテゴリ。口座開設後の早期の定期預金を後押しするキャンペーンです。

  - **対象者:** 期間中に新規口座を開設した方、または口座開設後まだ取引のない方
  - **対象期間:** 2026年7月1日 ～ 2026年9月30日（預り入れ日期間）
  - **達成条件と特典（上限あり）:**

  | 達成条件 | キャッシュバック |
  |----------|------------------|
  | 円定期預金(3か月)に合計200万円以上、満期まで保有すること | 預金額100万円ごとに1,000円 |
  | **合計上限** | **20,000円** |

  - **条件確認タイミング:** 判定日は満期となる月の翌月の1日で、1日は休日・祝日の場合にその次の最初の営業日である。
  - **特典付与時期:** 定期預金が満期となる月の翌月の10日までに、対象口座へ特典を現金として振込を行う。
  - **併用:** CMP-DEP-2026Q3-01キャンペーンと併用不可。
  - **関連商品:** [預金](products/Deposits.md)

### CMP-LOAN-2026Q3-01 — カードローン新規ご契約クーポンプレゼント
  カードローンカテゴリ。新規ご契約と初回のご利用に対してクーポン券を献呈します。

  - **対象者:** 期間中にカードローンを新規契約し、初回借入を行った方
  - **対象期間:** 2026年7月1日 ～ 2026年9月30日
  - **達成条件:**
    - カードローンを新規に契約
    - 契約後60日目の時点で借入残高が10万円以上
  - **特典:** 提携店で使えるクーポン券、確認日時点の借入残高10万円ごとに500円分を献呈
  - **利用できる店舗:** 日本国内の主なコンビニストア、ショッピングセンター、ドラッグストア、書店、家電量販店で利用可能
  - **併用:** 他のカードローン特典との併用不可
  - **合計上限:** クーポン券上限は25,000円
  - **条件確認タイミング:** 新規契約後の60日目、該当日が休日・祝日の場合は、次の最初の営業日が確認日である。確認日時点の借入残高をもとに達成条件を判断
  - **特典付与時期:** 条件達成を確認した月の翌月の10日まで(電子クーポン)。お客様がアプリでクーポンを検索・表示し、店頭のレジへ提示、利用
  - **クーポンの有効期限:** クーポン券面で明記。通常は付与期日から3年間まで有効。
  - **注意:** カードローンのご契約には当社および保証会社の審査があります。
    審査結果によりご契約・ご希望額に添えない場合があります（[カードローン](products/Lending.md) 参照）。

  - **関連商品:** [カードローン](products/Lending.md)

### CMP-DEBIT-2026Q3-01 — デビットカード新規発行・ご利用キャッシュバック
  デビットカードカテゴリ。新規申請・発行後の早期利用に対してキャッシュバックします。

  - **対象者:** 期間中にデビットカードを新規申請・発行した方
  - **対象期間:** 2026年7月1日 ～ 2026年9月30日
  - **達成条件と特典（段階制）:**

  | 発行後90日以内の合計利用額 | キャッシュバック |
  |----------------------------|------------------|
  | 10万円以上 | 1,000円 |
  | 30万円以上 | 3,500円（上記に代えて） |

  - **利用条件:** 税金・公共料金の支払い、電子マネーチャージ、キャッシング、およびATM出金は、合計利用額の対象外とする
  - **返品・キャンセル:** 返品・キャンセルで発生した金額は、利用金額から差し引く
  - **再発行:** 紛失等でカードを再発行し、カード番号が変わっても、同一のキャンペーン参加として継続する。元のカードでの利用額、および発行日（発行後90日の集計の起点）は、再発行後のカードに引き継ぐ
  - **国際利用:** 日本国外の加盟店利用も対象とする。ただし、支払い金額は最終換算後の日本円単位で計算
  - **条件確認タイミング:** カード発行後の90日目までの利用金額を、90日目以降の2週間以内で確認。
  - **特典付与時期:** カード発行後の104日目(90日+14日)の所在月、その翌月の10日までに新規発行のデビットカードに紐付けた口座に振り込む。
  - **ポイント加算:** 通常のポイント還元（0.6%）に加えて進呈
  - **併用:** 通常のポイント還元と併用可
  - **関連商品:** [デビットカード](products/Debit-card.md)

### デモにおける利用（参考）
  各キャンペーンは `CMP-<商品>-<時期>-<連番>` の ID を持ち、プッシュ通知のディープリンクは
  この ID をコンシェルジュのセッションに渡します。コンシェルジュは当該キャンペーンと
  お客さま情報をもとに、条件達成のシミュレーション（例: 定期預金の利息、デビットの
  キャッシュバック額）や申込の下書き保存を案内します。

> 想定シナリオはペルソナ集を参照: [Personas](Personas.md)。
