## キャンペーン（Campaigns）

> OneNeo Bank コンシェルジュ・デモ用のキャンペーン・プロファイル（架空・説明用）。
> 会社概要: [OneNeo-Bank](OneNeo-Bank.md) ／ 商品概要: [Products](Products.md)。
> 関連商品: [預金](products/Deposits.md) ／ [カードローン](products/Lending.md) ／
> [デビットカード](products/Debit-card.md)。
> 金額・条件・期間などの数値はすべて説明用であり、自由に改訂可能です。

OneNeo Bank のキャンペーンは、いずれも**口座開設やデビットカードの新規申請直後の
早期利用**を促すことを狙いとしています。お客さまはアプリのプッシュ通知でキャンペーンを
知り、通知をタップしてコンシェルジュに相談したり、条件のシミュレーションや申込の
下書き保存を行います。

キャンペーンは次の3カテゴリです。

1. **預金** — 普通預金・定期預金で、一定条件の達成によりキャッシュバック
2. **カードローン** — 新規契約で、一定条件の達成によりクーポン券を献呈
3. **デビットカード** — 新規申請＋利用で、一定条件の達成によりキャッシュバック

### 共通事項
  - **対象者:** 日本国内在住の個人のお客さま（各キャンペーンの対象条件を満たす方）
  - **エントリー:** アプリのキャンペーン画面からエントリー（事前エントリー制）
  - **特典付与:** 条件達成を確認後、原則として翌月末までに普通預金口座へ入金／付与
  - **併用:** 特典の併用可否は各キャンペーンに記載
  - **除外:** 反社会的勢力に該当する方、および当社が不適切と判断した方は対象外
  - 内容は予告なく変更・終了する場合があります

### CMP-DEP-2026Q3-01 — 口座開設・普通預金応援キャッシュバック
  預金カテゴリ。口座開設後の早期のお取引を後押しするキャンペーンです。

  - **対象者:** 期間中に新規口座を開設した方、または口座開設後まだ取引のない方
  - **対象期間:** 2026年7月1日 ～ 2026年9月30日
  - **達成条件と特典（複数達成可・合計上限あり）:**

  | 達成条件 | キャッシュバック |
  |----------|------------------|
  | 給与または年金の受取口座に設定 | 1,000円 |
  | 口座開設後60日以内に普通預金を50万円以上預入 | 1,000円 |
  | 3か月連続で普通預金の月末残高100万円以上を維持 | 2,000円 |
  | **合計上限** | **4,000円** |

  - **併用:** 上記条件は併用可（合計上限の範囲内）
  - **関連商品:** [預金](products/Deposits.md)

### CMP-DEP-2026Q3-02 — 口座開設・定期預金支援キャッシュバック
  預金カテゴリ。口座開設後の早期の定期預金を後押しするキャンペーンです。

  - **対象者:** 期間中に新規口座を開設した方、または口座開設後まだ取引のない方
  - **対象期間:** 2026年7月1日 ～ 2026年9月30日
  - **達成条件と特典（上限あり）:**

  | 達成条件 | キャッシュバック |
  |----------|------------------|
  | 円定期預金(3か月から1年まで)に合計200万円以上、満期まで保有すること | 預金額100万円ごとに1,000円 |
  | **合計上限** | **20,000円** |

  - **併用:** 対象外
  - **関連商品:** [預金](products/Deposits.md)

### CMP-LOAN-2026Q3-01 — カードローン新規ご契約クーポンプレゼント
  カードローンカテゴリ。新規ご契約と初回のご利用に対してクーポン券を献呈します。

  - **対象者:** 期間中にカードローンを新規契約し、初回借入を行った方
  - **対象期間:** 2026年7月1日 ～ 2026年9月30日
  - **達成条件:**
    - カードローンを新規に契約
    - 契約後60日以内に10万円以上の借入
  - **特典:** 提携店で使えるクーポン券、10万円ごとに500円分を献呈
  - **併用:** 他のカードローン特典との併用不可
  - **合計上限:** クーポン券上限は25,000円
  - **注意:** カードローンのご契約には当社および保証会社の審査があります。
    審査結果によりご契約・ご希望額に添えない場合があります（[カードローン](products/Lending.md) 参照）。

  - **関連商品:** [カードローン](products/Lending.md)

### CMP-DEBIT-2026Q3-01 — デビットカード新規発行・ご利用キャッシュバック
  デビットカードカテゴリ。新規申請・発行後の早期利用に対してキャッシュバックします。

  - **対象者:** 期間中にデビットカードを新規申請・発行した方
  - **対象期間:** 2026年7月1日 ～ 2026年9月30日
  - **達成条件と特典（段階制）:**

  | 発行後90日以内の合計利用額 | キャッシュバック |
  |----------------------------|------------------|
  | 10万円以上 | 1,000円 |
  | 30万円以上 | 3,500円（上記に代えて） |

  - **付与:** 通常のポイント還元（0.6%）に加えて進呈
  - **併用:** 通常のポイント還元と併用可
  - **関連商品:** [デビットカード](products/Debit-card.md)

### デモにおける利用（参考）
  各キャンペーンは `CMP-<商品>-<時期>-<連番>` の ID を持ち、プッシュ通知のディープリンクは
  この ID をコンシェルジュのセッションに渡します。コンシェルジュは当該キャンペーンと
  お客さま情報をもとに、条件達成のシミュレーション（例: 定期預金の利息、デビットの
  キャッシュバック額）や申込の下書き保存を案内します。

> 想定シナリオはペルソナ集を参照: [Personas](Personas.md)。
